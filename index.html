信息的底层表示（二进制）

二进制 (Binary)：计算机底层仅通过 0 和 1（即比特 Bit）处理信息。

单位转换：8 Bits = 1 Byte（字节）。

文本：通过 ASCII 或 Unicode（支持 Emoji）将数字映射为字符。

颜色：通过 RGB 模型（红、绿、蓝三种颜色的数值比例）表示图像。

计算机科学的本质

核心逻辑：输入 (Input) → 算法 (Algorithm) → 输出 (Output)。

算法：解决问题的精确、分步指令。

伪代码 (Pseudocode)：用简洁的人类语言描述程序逻辑，不涉及具体语法。

编程逻辑核心（Scratch 概念）

通过 Scratch 的积木块，理解所有编程语言通用的基础：

函数 (Functions)：执行动作的指令（如：移动、说话）。

循环 (Loops)：重复执行代码块。

条件 (Conditionals)：根据判断做出分支（If-Else）。

布尔表达式 (Boolean)：只有“真”或“假”的判断条件。

变量 (Variables)：存储和更新数据的容器。

事件 (Events)：触发程序运行的信号（如：当绿旗被点击时）。

多线程 (Threads)：多个任务同时进行。
